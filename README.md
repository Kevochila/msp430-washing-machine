# 🧺 MSP430 Washing Machine Controller

Proyecto desarrollado en MSP430 que simula el funcionamiento de una lavadora automática mediante:

- Control de motor a pasos (28BYJ-48)
- Interfaz con LCD 16x2 (modo 4 bits)
- Botones con interrupciones (FSM)
- Control de ciclos: remojado, lavado y exprimido

## ⚙️ Características

- Máquina de estados (FSM)
- Librerías modulares (LCD + Stepper)
- Antirrebote por hardware (RC)
- Control de velocidad

## 🔌 Hardware

- MSP430G2553
- LCD 1602A
- Motor paso a paso 28BYJ-48
- Driver ULN2003
- Botones + capacitores
